## Udemy Clone Website

This project is a **basic Udemy-inspired website clone** created using **HTML and CSS**. The purpose of this project is to practice front-end web development fundamentals such as creating web page layouts, navigation, and styling.

The website includes a **Home page** that displays course-related content similar to an online learning platform and a **Contact page** where users can reach out. This project helped me understand how to structure web pages and design simple user interfaces using HTML and CSS.

### Pages

* **Home Page** – Displays the main layout with course sections and website navigation.
* **Contact Page** – Contains a simple contact form or contact information.

### Technologies Used

* **HTML5** – For creating the structure of the website
* **CSS3** – For styling and layout design

### Purpose of the Project

This project was built to improve my understanding of **basic web development concepts** and to practice building a simple multi-page website.
